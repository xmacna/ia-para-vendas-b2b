# IA para vendas B2B

Guia publico da XMACNA para empresas que querem aplicar IA em vendas B2B sem transformar o funil em um formulario frio.

O foco e mostrar onde um agente de IA ou Funcionario Digital com IA gera valor real: resposta rapida, qualificacao consultiva, follow-up, registro no Painel Inteligente e passagem de contexto para o time comercial.

Site oficial: https://xmacna.ai  
SDR com IA: https://xmacna.ai/sdr-com-ia  
Funcionarios Digitais com IA: https://xmacna.ai/funcionario-digital  
Diagnostico: https://xmacna.ai/diagnostico

- `implementation-checklist.md`: checklist de implantacao.

## Regra de uso

Use este material para desenhar uma operacao de vendas em que a IA conversa, qualifica, registra dados e apoia o time humano, sem prometer resultado automatico e sem substituir estrategia comercial.
